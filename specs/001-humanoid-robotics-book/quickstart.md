# Quickstart Guide

This guide provides the basic steps to set up the development environment for following along with the book.

## System Requirements

- **Operating System**: Ubuntu 22.04 LTS
- **Hardware**:
  - **CPU**: Intel Core i7 or AMD Ryzen 7 (or equivalent)
  - **RAM**: 32 GB
  - **GPU**: NVIDIA RTX 3070 or better (for NVIDIA Isaac Sim)
  - **Storage**: 100 GB of free space

## Software Installation

1.  **Install ROS 2 Humble**: Follow the official ROS 2 documentation to install ROS 2 Humble Hawksbill.
2.  **Install Gazebo 11**: Install Gazebo 11, which is the recommended version for ROS 2 Humble.
3.  **Install Unity Hub**: Install Unity Hub and then install Unity 2022.3 LTS.
4.  **Install NVIDIA Isaac Sim**: Follow the instructions on the NVIDIA website to install Isaac Sim 2023.1.
5.  **Install Docusaurus**: If you want to build the book from source, you will need to install Node.js and then install Docusaurus.

## Cloud-Based Environment

For readers who do not have the required hardware, a dedicated chapter in the book will explain how to set up a cloud-based development environment using services like AWS or Google Cloud.
