# Research for "Physical AI & Humanoid Robotics Book"

This document records the decisions made regarding the tools and technologies used in the book.

## Tool Versioning

- **ROS 2**: The book will use **ROS 2 Humble Hawksbill**, as it is the latest long-term support (LTS) release and has a large community and good documentation.
- **Gazebo**: The book will use **Gazebo 11**, as it is the version recommended for ROS 2 Humble.
- **Unity**: The book will use **Unity 2022.3 LTS**, as it is a stable, long-term support version suitable for robotics simulation.
- **NVIDIA Isaac Sim**: The book will use **NVIDIA Isaac Sim 2023.1**, as it is the latest stable release with the most up-to-date features for robotics simulation and AI training.
- **Docusaurus**: The book will use **Docusaurus 3**, as it is the latest major version with modern features for building documentation websites.
- **Ubuntu**: The code examples will be developed and tested on **Ubuntu 22.04**, which is the recommended OS for ROS 2 Humble.
- **Python**: The Python code will be written for **Python 3.10**, which is the default Python version in Ubuntu 22.04.

## Deployment

- The book will be deployed as a static website on **GitHub Pages**. This decision was made based on the project's constitution, which requires the content to be optimized for Docusaurus and GitHub Pages.
